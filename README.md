### 🎯 계획
- 선정된 한 사람이 **서술형 방식의 시험 문제** 준비
    - 시험 문제는 무조건 서술형
    - 출제자는 사전 학습 후 핵심 주제에 대해 시험 문제를 출제
        - 문제 수 : 5문제
        - 시험 시간 : 30 - 40분 (출제자가 판단)
- PR을 통해 풀이를 올린다.
- 시험 시간 종료 후 해설 PR을 확인하고 각자의 정답을 발표 후 공유 피드백 진행
